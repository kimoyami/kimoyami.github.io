2015 ACM-ICPC Asia Regional Contest Shanghai Site-I(Infinity-Point-Sets)

发表于 2019-09-23 | 分类于 计算几何 | | 浏览 次

链接：http://acm.hdu.edu.cn/showproblem.php?pid=5581
思路：通过画图观察发现，四个点以下都是可以的，我们只用考察五个点及其以上的情况。发现最坏的情况是一条线加上两侧各一个点，再多就不行了。那么我们需要统计的就是：
1、三点共线及以上 + 两侧各一个点。
2、四点共线及以上 + 非线上任意一点。
3、五点共线。
为了不重复，我们考虑枚举每个点后，只处理在他上方的（y值相等必须x大于），但我们仍然需要对所以处理，因为外面选点是对于所有点而言的。这样总的情况上不会重复计算，但是这样可能出线下图额外算重的情况:

我们只需要在上半部分旋转的时候统计一下直线两个方向分别的点数，然后组合数去重以下即可。

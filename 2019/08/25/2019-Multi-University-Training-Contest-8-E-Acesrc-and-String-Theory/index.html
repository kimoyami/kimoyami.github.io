2019 Multi-University Training Contest 8-E(Acesrc and String Theory)

发表于 2019-08-25 | 分类于 后缀数组 | | 浏览 次

链接：http://acm.hdu.edu.cn/showproblem.php?pid=6661
思路：求是某个子串k次重复的子串有多少个。考虑这种免不了要枚举循环节。当循环节确定后，我们枚举某一个起点x，然后看它与x + len的lcp是否能出现k次，如果能我们可以前后拓展，这样整个sa数组就被分为了几段，每一段里面任意连续k * len长度的串都满足条件，这样我们对于每一段都可以统计贡献，所以我们需要正着反着都做一次SA，然后找到一段len向前后能拓展的最大长度，然后统计答案，之后跳到最右端开始新的一段。由于每次跳至少为len，根据调和级数可以知道复杂度最坏为O(nlogn)
